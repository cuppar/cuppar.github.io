Perl

Perl (Practical Extraction and Report Language)

实用报表提取语言

特性

  • 高级、通用、直译、动态、
  • 借用C、sed、awk、shell脚本等语言特性
  • 内部集成了正则表达式、巨大的第三方代码库CPAN

Hello World

第一行#!/usr/bin/perl为perl解释器的路径。
第二行print "Hello,World!\n";为打印字符串。

Tips

  • perl代码以“;”结尾，刚开始编写容易与Python混淆，老是忘记分号。
  • 标识符大小写敏感。
  • print可以用括号也可不用。
  • 字符串可用双引号或单引号，双引号解析转义字符和变量，单引号原样输出字符串。

Date Type

  • scalar(标量)
    $开头、可表示数字、字符串、引用、（Perl只有浮点数）

  • array(数组)

  • hash(哈希)
